# 📊 DRE Profissional - Confeitaria

## Sobre o Projeto

Este projeto apresenta uma Demonstração do Resultado do Exercício (DRE) desenvolvida no Microsoft Excel.

O objetivo foi simular a análise financeira de uma confeitaria de sucesso, utilizando uma estrutura semelhante às empregadas por empresas nas áreas de:

- Controladoria
- FP&A (Financial Planning & Analysis)
- Finanças Corporativas
- Gestão Financeira

A planilha possui cálculos automáticos, indicadores financeiros e análise mensal durante um período de 12 meses.

A planilha está organizada nas seguintes abas:

Contém a Demonstração do Resultado do Exercício com:

- Receita Bruta
- Receita Líquida
- Custos
- Lucro Bruto
- EBITDA
- EBIT
- Resultado Financeiro
- Lucro Líquido

Apresenta indicadores financeiros para auxiliar a análise do desempenho da empresa.
